# BiomassBro

BiomassBro is a web-based platform designed for the calculation and analysis of biomass data. It allows researchers and environmental analysts to estimate biomass value in chosen area and visualize data through an interactive dashboard.

## Live Application
The application is deployed and can be accessed at:
https://biomassbro.streamlit.app/
